## Setup

### Clone repo

```bash
git clone https://github.com/devmoamal/LiteMarkdownEditor
cd LiteMarkdownEditor
```

### Install packages

```bash
npm install
```

### Run dev server

```bash
npm run dev
```

## Technologies

- Vite
- ReactJs
- Typescript
- Tailwindcss v4
- Zustand
